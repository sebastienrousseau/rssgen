pub fn generate_rss(options: &RssData) -> Result<String>
Expand description

Generates an RSS feed from the given RssData struct.

This function creates a complete RSS feed in XML format based on the data contained in the provided RssData. It generates the feed according to the RSS version set in the RssData.
